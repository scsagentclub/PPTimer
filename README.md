# PPT-Timer
A lifetime-free PowerPoint countdown timer software.

1、软件不用安装，放置桌面点击即可运行
2、+－或滚轮可调整时间
3、点击logo可重置时间
4、PPT全屏放映可自动倒计时
5、点击时间可启动/暂停倒计时
6、按esc退出
BY：sunnybluesea

1. No installation required; simply place on your desktop and click to run.

2. Use the +/- keys or scroll wheel to adjust the time.

3. Click the logo to reset the time.

4. Automatic countdown during full-screen PowerPoint presentation.

5. Click the time to start/pause the countdown.

6. Press Esc to exit.
